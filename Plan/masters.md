# 📋 Course-to-Category Mapping (with Blocks)

### 🧑‍💻 Programming

*(e.g., object-oriented programming, functional programming, software architecture)*

*   **Block 1A** — Introduction to Programming with Python — **3 ECTS**
*   **Block 1D** — Advanced Programming Concepts I — **3 ECTS**
*   **Block 2D** — Advanced Programming Concepts II — **5 ECTS**

**Subtotal (Programming): 11 ECTS**

> **Note (possible additions if syllabi emphasize implementation/software design):**
>
> *   **Block 1D** — Data Engineering I (3 ECTS)
> *   **Block 2D** — MLOps & Cloud Engineering (5 ECTS), Data Engineering II (5 ECTS)
> *   **Block 1A** — Introduction to Data Science (3 ECTS)

***

### 🖥️ Computer Systems

*(e.g., databases, computer architecture, networks, operating systems, distributed systems, security)*

*   **Block 1B** — Introduction to SQL — **2 ECTS** *(Databases)*
*   **Block 1D** — Data Engineering I — **3 ECTS** *(Databases / data pipelines)*
*   **Block 2D** — Data Engineering II — **5 ECTS** *(Databases / data pipelines)*
*   **Block 2D** — MLOps and Cloud Engineering — **5 ECTS** *(Distributed systems / cloud deployment)*

**Subtotal (Computer Systems): 15 ECTS**

> **Note (possible addition if systems topics are covered):**
>
> *   **Block 1A** — Introduction to Data Science (3 ECTS)

***

### 🧠 Theoretical Computer Science

*(e.g., algorithms & data structures, logic & computability, formal languages & compilers, optimization, complexity theory, **and machine learning**)*

*   **Block 1A** — Introduction to AI — **5 ECTS**
*   **Block 1B** — Introduction to Machine Learning — **9 ECTS**
*   **Block 1C** — Deep Learning for Image Classification — **6 ECTS**
*   **Block 2A** — Natural Language Processing — **15 ECTS**
*   **Block 2B** — Computer Vision — **8 ECTS**
*   **Block 2B** — Robotics and Reinforcement Learning — **7 ECTS**
*   **Block 1C** — Responsible & Explainable AI — **4 ECTS** *(often accepted here; some schools view it as ethics/HCI depending on focus)*

**Subtotal (Theoretical CS): 54 ECTS**

***

### 🧑‍🤝‍🧑 Human–Computer Interaction

*(e.g., interaction design, experimental system development)*

*   **Block 1C** — Human-Centered AI — **3 ECTS**

**Subtotal (HCI): 3 ECTS**

> **Note:** If *Responsible & Explainable AI (Block 1C, 4 ECTS)* includes user studies, interpretability UX, or interactive explanations, it can sometimes be argued under **HCI** as well.

***

### 🔢 Mathematics

*(e.g., calculus, linear algebra, probability theory, statistics)*

*   **Block 1A** — Foundational Mathematics — **4 ECTS**
*   **Block 1B** — Mathematical Foundations for Machine Learning — **4 ECTS**
*   **Block 1D** — Time Series Analysis — **7 ECTS**
*   **Block 2C** — Statistics — **5 ECTS**

**Subtotal (Mathematics): 20 ECTS**
